# 尚品汇

#### 介绍
尚品汇电商平台  ，总功能使用vue为框架，vue-router进行路由的跳转，vuex仓库进行数据的储存。最后映射至组件完成数据动态展示。后台数据的获取是使用的是尚硅谷提供的开源api。完成了商品的展示，搜索，详情功能，其余因为时间问题还待完善。通过主页左侧的选项卡点击‘手机’进入详情页，展示相应的参数。点击相应产品进入产品介绍与加入购物车等功能。搜索功能类似。
